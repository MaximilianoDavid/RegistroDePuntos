
# Registro de puntos 🔫

El algoritmo simula un torneo de un videojuego de disparos (Shooter) en el que participan varios jugadores. 

Su función principal es registrar el desempeño de cada jugador, calcular sus puntajes y determinar quién obtuvo la mejor puntuación del torneo.

Durante cada partida, el usuario debe ingresar la cantidad de kills (eliminaciones) obtenidas. 

A partir de este valor se calcula el puntaje de la partida multiplicando las kills por 150 puntos. 

Además, el algoritmo otorga bonificaciones según el rendimiento alcanzado:



## Puntuación 🏆

• 15 o más kills: +500 puntos.

• Entre 10 y 14 kills: +250 puntos.

• Entre 5 y 9 kills: +125 puntos.

• Menos de 5 kills: sin bonificación.
